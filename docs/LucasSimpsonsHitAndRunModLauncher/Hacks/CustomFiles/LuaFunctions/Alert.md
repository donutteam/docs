Shows a Windows message box with an "OK" button for debugging purposes.

# Syntax
```lua
Alert( message )
```

## Arguments
* **message** (string): The message to show in message box.

## Return Values
No return values.

# Examples
```lua
Alert("This is an alert message from a mod.")
```
