The `Session` table provides functions for managing server sessions in Simpsons Hit & Run Multiplayer Server mods.

This functionality is extremely early in development and is subject to change.

## Variables
This table does not have any variables.

## Methods
| Function Name              | Description                                                     |
|----------------------------|-----------------------------------------------------------------|
| [[Session/GetMainMods.md]] | Retrieves all main mods currently loaded in the server session. |
