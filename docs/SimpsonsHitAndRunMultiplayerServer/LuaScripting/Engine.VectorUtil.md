The `Engine.VectorUtil` table provides utility functions for working with vectors.

## Variables
This table does not have any variables.

## Methods
| Function Name                           | Description                                                                                    |
|-----------------------------------------|------------------------------------------------------------------------------------------------|
| [[Engine.VectorUtil/GetDistance2D.md]]  | Returns the distance between two points in 2D space.                                           |
| [[Engine.VectorUtil/WithinDistance.md]] | Returns true if the distance between two points is less than or equal to a specified distance. |
| [[Engine.VectorUtil/InSphere.md]]       | Returns true if a point is inside a sphere.                                                    |
