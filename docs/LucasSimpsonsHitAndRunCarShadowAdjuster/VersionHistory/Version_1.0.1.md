**This version was released on January 25, 2021.**

* Added support for the Best Sellers Series, International and Demo versions of the game.
* Added an **Open...** option to the File menu.
* Added icons to the File menu.
* Made the tool support any filename for the game executable instead of just "Simpsons.exe".
* Changed the copyright date from 2003-2014 to 2014-2015.
* Changed the error message shown when the game is not running to have a colon in "The Simpsons Hit & Run".
* Made it so you can use the backspace key on the numeric textboxes.

# Notes
This update was ready to release on October 30th, 2015 and we forgot to actually release it until January 26th, 2021. Fascinating.
